# features

- **KernelSU (fork)**: KernelSU is a root solution for Android GKI devices, it works in kernel mode and grants root permission to userspace applications directly in kernel space. in this kernel im using ksun fork - wildksu
- **SUSFS**: an addon root hiding kernel patches and userspace module for ksu(n).
